`ps`
====
Allows to view running Flux processes

Syntax
----
```
ps [-h] [mode]
```

Positional Arguments: | Description
--------|------------
`mode` | How to view the processes <(s)imple|(l)ist|(o)utput|(t)hreads|(m)isc|(a)ll>

Options | Description
--------|------------
`-h, --help` | Show this help message and exit
